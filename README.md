🎉 Lottie Animations: Ilovani jonlantiruvchi, foydalanuvchiga hissiy aloqa beruvchi dinamik vektor animatsiyalar foni.

📖 ViewPager (Swipe): Mavzular bo'ylab silliq, kitob varaqlash kabi intuitiv harakatlanish (Fluid UX).

🖼️ Fresco Engine: Ranglar va rasmlarni ultra-tezkor renderlash orqali vizual jozibadorlikni ta'minlash.

💾 Smart Persistence: JSON orqali mavzular bazasini va SharedPreferences orqali erishilgan ballarni xavfsiz saqlash.
